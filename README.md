# Elegant Portfolio Website

A modern, responsive, and bilingual portfolio website built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

The website presents personal information, selected projects, work experience, and contact links with smooth animations and an interface optimized for desktop and mobile devices.

## Features

- Indonesian and English language support
- Responsive design
- Animated backgrounds and transitions
- Project and work experience sections
- GitHub, LinkedIn, YouTube, and email contact links

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Motion
- Three.js
